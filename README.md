# InterviewPresence AI

InterviewPresence AI is an AI-powered mock interview platform that simulates real interview environments and evaluates candidates using computer vision, speech analysis, and NLP-based semantic scoring.

The system supports Technical, HR, and Behavioral interviews with performance analytics, history tracking, and interactive feedback.

# AI Workflow

      ↓
Video Analysis (Face + Eye Contact)
      ↓
Audio Analysis (Speech Activity)
      ↓
Speech-to-Text Conversion
      ↓
Semantic Similarity Scoring
      ↓
Final AI Evaluation Score
